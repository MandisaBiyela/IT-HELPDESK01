```python
def calculate_sla_deadline(priority: TicketPriority):
    if priority == URGENT:
        return created_at + timedelta(minutes=20)
    elif priority == HIGH:
        return created_at + timedelta(minutes=480)  # 8 hours
    else:  # NORMAL
        return created_at + timedelta(minutes=1440)  # 24 hours
```
